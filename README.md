# 🍽️ ADBU_CANTEEN - Canteen Management System

Overview
ADBU_CANTEEN is a web-based canteen management system developed as a university project for Assam Don Bosco University. The application aims to simplify and digitize canteen operations by providing an interactive platform for both users and administrators.

---

Technologies Used

Frontend
- HTML
- CSS
- JavaScript

 Backend
- PHP

Database
- MySQL

---

 Key Features

 User Side
-  View available food menu
-  Submit reviews and feedback
-  Responsive and user-friendly interface

Admin Panel
 Add new menu items
 Delete menu items
 Manage canteen data dynamically



 Database Functionality
- Stores menu details
- Saves user reviews directly into the database
- Enables dynamic interaction between frontend and backend
